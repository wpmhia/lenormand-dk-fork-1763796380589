# SEO Resources & Tools - Quick Reference

## Essential Google Tools (Free)

### 1. Google Search Console
**URL:** https://search.google.com/search-console/
**Purpose:** Monitor search performance, submit sitemap, fix errors
**Key Actions:**
- Submit sitemap.xml
- Monitor keyword rankings
- Check indexing status
- Fix crawl errors
- View search analytics

### 2. Google Analytics 4
**URL:** https://analytics.google.com/
**Purpose:** Track user behavior and conversions
**Key Metrics:**
- Organic traffic source
- User engagement (time on site, bounce rate)
- Conversion tracking (reading completions)
- Device breakdown (mobile vs desktop)

### 3. Google Pagespeed Insights
**URL:** https://pagespeed.web.dev/
**Purpose:** Analyze Core Web Vitals and page speed
**Key Metrics:**
- LCP (Largest Contentful Paint)
- FID (First Input Delay)
- CLS (Cumulative Layout Shift)
- Overall score (target: 90+)

### 4. Google Lighthouse
**Access:** Chrome DevTools → Lighthouse tab
**Purpose:** Comprehensive page audit
**Tests:**
- Performance
- Accessibility
- Best practices
- SEO
- PWA readiness

### 5. Google Structured Data Testing Tool
**URL:** https://schema.org/validator
**Purpose:** Validate JSON-LD schemas
**Test:** Paste your JSON-LD and verify syntax

## SEO Metrics Cheat Sheet

```
On-Page SEO:
- Title length: 50-60 characters
- Meta description: 150-160 characters
- URL length: <75 characters
- H1 count: 1 per page
- Keyword density: 1-2%

Technical SEO:
- Page load time: <3 seconds
- Mobile score: 90+
- Desktop score: 90+
- Core Web Vitals: All "Good"

Authority:
- Domain Authority: 30+
- Page Authority: 20+
- Backlinks: 50+ quality links
- Referring domains: 20+

Traffic:
- Month 1: 100-200 visits
- Month 3: 500-1,000 visits
- Month 6: 2,000-5,000 visits
```
